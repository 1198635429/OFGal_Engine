# 蓝图节点类型

我们的节点结构体的引脚数组将根据类型来初始化，故列出所有类型的节点应该要有什么引脚

## 事件函数Event_
### BeginPlay
在蓝图开始运行时
```json
"pins" : [
{"name" : "OEXEC","I/O" : "O","type" : "exec"}
]
```
### FuncPlay
在蓝图作为函数被其它蓝图调用时
```json
"pins" : [
{"name" : "OEXEC","I/O" : "O","type" : "exec"},
{"name" : 用户自定义的传入参数名,"I/O" : "I","type" : 用户自定义的传入参数类型},
//...
]
```
### Tick
当蓝图开始运行时每秒触发50次该节点
```json
"pins" : [
{"name" : "OEXEC","I/O" : "O","type" : "exec"}
]
```
### Custom
用户自定义的事件类型，当被触发时调用该节点
```json
"pins" : [
{"name" : "OEXEC","I/O" : "O","type" : "exec"}
]
```
## 控制函数
### if
条件判断函数，同if
```json
"pins" : [
{"name" : "IEXEC","I/O" : "I","type" : "exec"},
{"name" : "Condition","I/O" : "I","type" : "bool"},
{"name" : "true","I/O" : "O","type" : "exec"},
{"name" : "false","I/O" : "O","type" : "exec"}
]
```
### while
循环控制函数，同while
```json
"pins" : [
{"name" : "IEXEC","I/O" : "I","type" : "exec"},
{"name" : "Condition","I/O" : "I","type" : "bool"},
{"name" : "LoopBody","I/O" : "O","type" : "exec"}
]
```
### break
循环控制函数，同break，若在非循环中连接则蓝图转译报错
```json
"pins" : [
{"name" : "IEXEC","I/O" : "I","type" : "exec"},
{"name" : "OEXEC","I/O" : "O","type" : "exec"}
]
```
