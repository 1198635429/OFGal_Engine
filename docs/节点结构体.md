# 节点结构体
## 加减乘除
简单运算节点，输出两个数的运算结果
```cpp
Node {
	int id = ???
	std::string type = "Add"或"Sub"或"Mul"或"Div"
	std::vector<Pin> pins;	//5个
	std::map<std::string, std::string> properties;   // 空，该节点无高级功能
};
```
#### 执行入口
```cpp
Pin {
	std::string name = "IEXEC";
	std::string io = "I";
	std::string type = "exec"
	std::optional<std::string> literal;   // 空，不存在字面值
};
```
#### 操作数A
字面值的读取方式以引脚类型为准，比如类型是bool但是字面值是"hello"或"123"就判断为真，类型是int但是字面值是"123.756"就读为123（直接丢弃小数）
```cpp
Pin {
	std::string name = "A";
	std::string io = "I";
	std::string type = "int"或"float"或"string"或"bool"
	std::optional<std::string> literal;   // 可能存在字面值
};
```
#### 操作数B
```cpp
Pin {
	std::string name = "B";
	std::string io = "I";
	std::string type = "int"或"float"或"string"或"bool"	// 必定与A类型相同
	std::optional<std::string> literal;   // 可能存在字面值
};
```
#### 执行出口
```cpp
Pin {
	std::string name = "OEXEC";
	std::string io = "O";
	std::string type = "exec"
	std::optional<std::string> literal;   // 空，不存在字面值
};
```
#### 运算结果
```cpp
Pin {
	std::string name = "result";
	std::string io = "O";
	std::string type = "int"或"float"或"string"或"bool"	// 必定与A类型相同
	std::optional<std::string> literal;   // 不存在字面值
};
```

## 后处理FrameProcess
对一个帧数据（frame）进行后处理，不访问场景或对象信息
属性键值对存储了各种后处理的参数，参数之间以英文逗号分隔

参数释义在项目OFGal_Engine的CUDA_SpeedUp的FrameProcessing.h中可以找到，参数按顺序输入，以下展示的只是默认参数
（参数如果是小数，那么一定会在尾部有字母"**f**"）
（所有的true和false在该节点的属性键值对中都视为布尔值而不是字符串）
```cpp
Node {
	int id = ???
	std::string type = "FrameProcess"
	std::vector<Pin> pins;	//5个
	std::map<std::string, std::string> properties;   // 可能有以下的键值对
	//"Bloom" : "220.0f,0.8f,4,-1.0f"
	//"FXAA" : "0.166f,0.05f,8.0f,0.125f,0.0078f"
	//"SMAA" : "0.05f,4,true"
	//"LensDistortion" : "0.0f,0.5f,0.5f"
	//"ChromaticAberration" : "2.0f,0,0.5f,0.5f"
	//"Blur" : "3.0f,-1.0f,0"
	//"Sharpen" : "0.5f,2,-1.0f"
	//"FilmGrain" : "0.05f,1,true,0"
	//"Vignette" : "0.3f,0.6f,1.0f,0.5f,0.5f,1.0f"
	//"ColorCorrection" : "0.0f,1.0f,1.0f,1.0f,1.0f,1.0f,0.0f"
	//"ColorGrading" : "0,0.8f,1.0f,1.0f,1.0f"
};
```
#### 执行入口
```cpp
Pin {
```
